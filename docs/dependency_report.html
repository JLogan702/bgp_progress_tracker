How is this report calculated?

We scan all story tickets for dependency links (Jira's 'Blocks' relationships). These relationships help teams identify and resolve blockers early.

  • Blocked By: This ticket is waiting on another ticket to be completed first.
  • Blocking: This ticket is holding up another ticket from starting.

Use this report to resolve critical-path issues and ensure your team’s work isn’t blocked before sprint kickoff.
